- Colar: SHIFT + INSERT

Comandos Git
- Configuram usuario e email do programador

	git config --global user.name "Seu Nome"
	git config --global user.email "seu_email@qualquercoisa.com"

- Verifica resulta dos comandos anteriores

	git config --list

- git status
- git diff
- git add
- git commit 
- git log
	- A opção --raw mostra o conteúdo dos commits
- git restore
	
Conceitos Git

- O comando "git init" cria um diretório oculto ".git" dentro do diretório de trabalho. Esse diretório ".git" é o repositório que mantém as fotografias(Snapshots) do nosso projeto.

- O arquivo .gitignore diz para o mecanismo do git quais arquivos ele deve ignorar, normalmente ele serve para indicar para o Git quais arquivos não são arquivos fontes, por exemplo, arquivos compilados(*.class, *.exe, etc.).
